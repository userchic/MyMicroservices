# MyMicroservices
Примитивный проект попытка сделать микросервисное приложение
# Суть проекта

Сделать простенькую соц сеть или что-то на подобие того где можно делать посты, редактировать и удалять. Возможно будет сделана подборка постов. И отдельно сервис аутентификации который будет передавать JWT токен с идентификатором пользователя чтобы другие сервис(ы) могли делать объекты со ссылкой на пользователя. Фронтенд работает с аутентификацией и после нее спрашивает с других сервисов - что у них есть.

## Состояние
На данный момент сделаны:
- Созданы 2 микросервиса на базе единственных сущностей. 
- Сделан GateWay с помощью YARP
- аутентификация по JWT. JWT призван предоставлять сервису постов в payload номер пользователя. 
- сделан фронтенд на React. Реализована отправка постов, редактирование и удаление и аутентификация с регистрацией.

Gateway был заменен на YARP, но проблема, что заголовок авторизации удалялся прокси видимо была связана не с Ocelot. Она решена путем добавления заголовка с собственным названием, судя по всему прокси стараются удалять какие-то заголовки и ищут нужные подстроки в заголовках.
