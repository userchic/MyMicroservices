# MyMicroservices
Примитивный проект попытка сделать микросервисное приложение

## Состояние
На жанный момент сделаны:
- Созданы 2 микросервиса на базе единственных сущностей. 
- Сделан GateWay с помощью Ocelot
- аутентификация по JWT. Аутентификация призвана предоставлять сервису постов в payload номер пользователя. 
- сделан фронтенд на React 
Пока что обнаружилось что Ocelot не получает токен в виде куки, а заголовок с токеном удаляет из за чего сервис постов не получает его. Возможно Ocelot будет заменен. Аутентификация работает.
